The topic of the project is "BattleBoard". The task was to create a new internet platform which will enable users to create their profiles and access features which will take playing board games to whole new level. Using the system enable to find thousands of games, create torunaments, invite friends, track time and points during the gameplay and competing to other players in the world through the ranking system. The application was created according to Single Page Application standards and is available here, also make sure to check out project repository and project Management along with the application prototype.
